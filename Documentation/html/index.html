Документация по Library Manager

Введение

Добро пожаловать в документацию по Library Manager! Эта документация предоставляет подробную информацию о функциях и классах Library Manager.

Назначение

Library Manager - это программное обеспечение, предназначенное для управления и организации коллекции книг библиотеки. Он предоставляет функциональность для добавления новых книг, удаления книг, поиска книг, а также для осуществления административных функций.

Использование

Использование происходит с помощью графического интерфейса.

Установка

Шаг 1: Скачайте пакет https://github.com/xdexix/LibraryManager.git удобным вам способом.

$ git clone https://github.com/xdexix/LibraryManager.git

Шаг 2: В зависимости от целевой платформы, произведите сборку (Необходим .NET7).

Windows: откройте файл .sln в VisualStudio.

Linux: войдите в директорию проекта и выполните запуск.

 $ cd LibraryManager
 $ dotnet run

Шаг 3: Если это необходимо, замените файл базы данных Data/DataBase.db.

$ cp [ваш файл] LibraryManager/Data/DataBase.db
