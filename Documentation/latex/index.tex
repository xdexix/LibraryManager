Library Manager -\/ это программное обеспечение, предназначенное для управления и организации коллекции книг библиотеки. Он предоставляет функциональность для добавления новых книг, удаления книг, поиска книг, а также для осуществления административных функций.\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Использование}{Использование}}\label{index_usage_sec}
